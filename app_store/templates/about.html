About VertiCapp

{% endblock %} {% block content %}

VertiCapp is an community application repository for the Vertica database system.

Vertica is an analytic SQL database designed for large, distributed queries. It's free for use up to 1 Terabyte of data. (Downloads here; sign up for a free Community account first.)

Vertica is highly extensible. It has a plugin architecture that allows anyone to implement new database operators, add new parse formats, even connect to other database systems. But installing plugins can be a pain.

That's where VertiCapps comes in. It provides a one-click (well, ok, one-command-line-command) installer for Vertica applications.

Why VertiCapp? Well, it used to be VerticApp. But that's hard to say. Then I found a picture of a monkey with a funky cap. So, VertiCapp.
